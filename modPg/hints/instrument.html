Server is not instrumented

$APPNAME implements some advanced features, which require additional procedures and tables in the maintenance database to work. These advanced features are:

  • aborting connections on the connection status page
  • Favorite objects in the browser tree
  • SQL code Snippets in the SQL query tool
  • Filter Presets in the data tool
Favorites, Snippets and Presets are stored in database tables personal to the user you're using to connect, and are instantly available on every computer you're connecting from using $APPNAME.

To prepare the server for improved handling through $APPNAME, execute the instrument context menu on the server object and close the connection. After re-opening the connection, the instrumentation state will be detected and the advanced features are available.

Note: The namespace used for configuration storage is configured in Preferences/PostgreSQL.
